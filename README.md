# _Anita Kemp's Salon Clientelle Tracker_

#### _A way to keep track of hair stylists and their clients as part of the code review for Epicodus, 10/18/19_

#### By _Anita Kemp_

## Description

This MVC application allows the user to input a list of hair stylists working at a salon along with their clients.

## Setup/Installation Requirements

* _Clone this repository_
* _In the command line, navigate to mysql (or use mysql workbench)_
* _CREATE DATABASE anita_kemp_
* _USE anita_kemp_
* _CREATE TABLE stylists (stylistId PRIMARY KEY, NN, AUTO INCREMENT, name VARCHAR(255))_
* _CREATE TABLE clients (clientId PRIMARY KEY, NN, AUTO INCREMENT, name VARCHAR(255), stylistId INT(11))_
* _Run dotnet watch run_
* _Navigate to localhost:5000/
